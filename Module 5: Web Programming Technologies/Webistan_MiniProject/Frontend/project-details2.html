Mutliliguistic Voice Controlled Wheelchair

Mutliliguistic Voice Controlled Wheelchair

Author: Samruddhi Desai

Technologies Used: Python,NLP

Languages Used: Python

Previous Next

The robot is equipped with advanced voice recognition technology that can understand and interpret commands in multiple languages, making it accessible to a global audience. Users can communicate with the robot in their preferred language.

Aim Of the Project
The aim of this project is to develop a multilinguistic voice-controlled wheelchair that can help wheelchair users to navigate wheelchair through voice commands in any language without need of external assistant.
Objectives
1. To develop a wheelchair that can seamlessly respond to voice commands using an embedded microphone, thereby eliminating the need for external control devices or applications.
2. To develop a robust voice recognition and processing system within the wheelchair, enabling it to interpret and execute a wide range of voice commands, such as movement, actions, and responses, with precision and efficiency.
3. To develop a multilingual system that supports diverse languages, to interact with autonomous robots.
